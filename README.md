# AI Face Detection

## Description

This project uses Python and OpenCV to detect faces in real time through a webcam. When a face is detected, the program draws a randomly colored rectangle around it.

## Requirements

* Python 3
* OpenCV (`cv2`)
* `haarcascade_frontalface_default.xml`

## How to Run

1. Install OpenCV:

   ```bash
   pip install opencv-python
   ```
2. Make sure `haarcascade_frontalface_default.xml` is in the project folder.
3. Run the program:

   ```bash
   python face_detection.py
   ```
4. Press **Q** to quit.
